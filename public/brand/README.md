# Brand Assets

Place the primary logo file at: `public/brand/logo/logo.png`.

## Naming conventions
- Use lowercase kebab-case for asset names.
- Include size or variant when needed: `logo-mark-48.png`, `icon-arrow-right-24.svg`.
- Prefer `.svg` for vector assets and `.png` for raster assets.

## Folder usage
- `logo/`: Primary and secondary logos (full lockups, marks, wordmarks).
- `icons/`: UI or feature icons used across the site.
- `favicons/`: Browser and device icon assets (favicon, apple touch icon, etc.).
